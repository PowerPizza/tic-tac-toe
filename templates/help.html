How To Create Game

Click on create game to create new game fill the details copy the id given in format(xxxxxxxx-xx...) and send it to the person with who you gonna play.

How to Join Game

Click on join fill the entry in 'ID to connect' put the ID that you recieved.

Replay

When a game finish then you see results and also at bottom of result you see a Replay option by click it the hoster of game can reset the game where connection ID will be same as old but chances can be change once hoster clicked on replay option then the you can join the game by click replay option.
